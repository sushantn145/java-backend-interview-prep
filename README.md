# java-backend-interview-prep
 Java
This repository is a structured learning and practice tracker for Java backend engineers.
It includes:
- Core Java & JVM internals
- Collections & concurrency
- Java 8+ (Streams, Optional, functional style)
- Multithreading & performance concepts
- Spring Boot best practices
- SQL & database fundamentals
- Low-level & high-level system design
- LeetCode problems with clean Java solutions
- Daily learning logs, weekly reviews, and interview mistake tracking

The goal is not just to learn tools, but to build strong fundamentals, design thinking, and interview confidence required for product-based companies.
